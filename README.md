# PR Review Service

### Стек:
 - Golang:1.25.4
 - Gin
 - Postgresql:15

### Команды для работы с проектом:

#### 1. Запуск docker-compose:
```bash
make up
```
#### 2. Просмотр логов контейнера приложения:
```bash
make logs
```
#### 3. Остановка docker-compose:
```bash
make down
```
#### 4. Отдельное создание контейнера приложения:
```bash
make docker-build
```
